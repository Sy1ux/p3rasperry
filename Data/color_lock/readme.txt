fonctionnement:

les menus:
-un menu aparait au lancement: 
	c'est qu'un fichier contenant un message est enregistré
	-rond: décrypter le message
	-triangle: réecrire un message
	-croix: suprimer le message

-un menu aparait aprés la rédaction d'un message:
	c'est pour demander confirmation du message
	-rond: enregistrer
	-triangle: retour a l'éditeur

-un menu aparait après lecture du message:
	c'est pour déterminer quois faire du message
	-rond: conserver
	-croix: suprimer

interface de creation de message (joystik):
-haut/bas: 
	naviguer a travers les caractères possibles
-gauche/droite: 
	naviguer a travers le message
-droite (depuis l'extrémitée du message): 
	crée un nouveau caractère au message
-milieu (double pression):
	suprime le dernier caractère du message
-milieu (pression longue):
	affiche le message complet et le menu de confirmation
